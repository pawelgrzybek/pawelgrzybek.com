## Top picks

### [Your URL Is Your State](https://alfy.blog/2025/10/31/your-url-is-your-state.html)

Often times we don’t need a complicated storage mechanisms, state management libraries or even database. Good, old URL is a beautiful way of passing state around, and reliably works on the web since 1991. I'm sure you heard that before but, please, use the platform devs.

### [The (lazy) Git UI You Didn't Know You Need](https://www.bwplotka.dev/2025/lazygit/)

It is no secret that [I'm a huge fan of lazygit](/lazygit-the-convenience-of-gui-with-the-power-of-git-cli/), and naturally I like to see more likeminded geeks embracing this powerful TUI. Bartek’s article highlights all the good reasons why you should try it, if you’re still pursuing the masochistic practice of using the git CLI. A little side note, that you should probably try Jujutsu, but this is a story for another day.
